2. Для чего НЕЛЬЗЯ использовать функцию "Фильтр" в окне управления слоями?

++ 		: Для группировки объектов по функциям
-- Для изменения параметров графики
-- Для организации совместной работы со смежным специалистом
-- Для разделения проекта по стадиям
-- Для группировки слоев по разделу проекта

Для чего МОЖНО применять фильтры слоев (анализ остальных вариантов):
*Для изменения параметров графики — выбрав фильтр (например, «Все слои вентиляции»), можно выделить их одной кнопкой и одновременно изменить цвет или вес линий для всей группы слоев.
*Для организации совместной работы — если чертеж делают несколько человек, фильтр позволяет отсечь чужие слои и видеть в списке только свои.
*Для разделения проекта по стадиям — можно настроить фильтрацию слоев для стадий «Демонтаж», «Проект» или «Существующие конструкции» на основе префиксов в их названиях.
*Для группировки слоев по разделу проекта — это классический сценарий, когда слои с именами «АР_...» (Архитектура) или «ОВ_...» (Отопление и вентиляция) группируются в отдельные папки-фильтры для удобства переключения.

4. К какому объекту геометрии НЕ может быть выполнена привязка в режиме "Середина"?

++ 		: Эллипс

-- Разность диаметров кольца
-- Отрезок 
-- Дуга
-- Старт построения прямой

*Старт построения прямой (XLINE / Бесконечная прямая) — когда вы только начинаете строить бесконечную прямую, первая указанная вами точка клика физически становится её базовой определяющей точкой (корнем). В AutoCAD эта точка программно считывается как «Середина» («Midpoint») данного геометрического элемента
*Разность диаметров кольца (DONUT) — кольцо в AutoCAD формируется из широкой полилинии, состоящей из двух полукруглых дуговых сегментов. Так как это дуговые сегменты, при наведении на внутреннюю или внешнюю грань кольца привязка «Середина» сработает на центральной точке одного из этих полукружий.

8. Что нужно изменить в указанном ниже порядке действий, чтобы успешно создать блок в AutoCAD?

++		: Функция создания блока недоступна через ПКМ - это можно сделать через вкладку "Блок"

-- Прежде чем создать блок, объекты необходимо объединить в группу
-- Не получится создать блок на текущем чертеже - их нужно продублировать на новый чертеж
-- Прежде чем выделить объекты и создать из них блок, необходимо перенести на их один слой
-- Ввести имя блока недостаточно - также необходимо добавить описание блока

* Выделите нужные объекты на чертеже.Нажмите ПКМ и выберите Буфер обмена -> Вырезать (или Копировать).Нажмите ПКМ в пустом месте чертежа.Выберите Буфер обмена -> Вставить как блок.
