Какое значение будет выведено в консоль при выполнении кода?
function testScope() {
	var x =7;
	if (true) {
		var x = 20;
		console.log(x);
	}
	console.log(x);
}
testScope();

правильный ответ:
++ 		: 20 20

ошибки:
-- 77
-- Ошибка: x не определена 
-- 20 7
-- 7 20

*В JavaScript переменные, объявленные с помощью ключевого слова var, имеют функциональную область видимости (function scope).Когда вы объявляете var x = 7; внутри функции, она становится видимой во всей функции testScope.Блок if в JavaScript не создает новую область видимости для var. Поэтому, когда внутри него выполняется var x = 20;, это не создание новой переменной, а изменение значения существующей x.
