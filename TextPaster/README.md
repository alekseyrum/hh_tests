
Расширение для Google Chrome, позволяющее скопировать в буфер обмена исходный текст практической задачи, для последующей вставки из буфера обмена в текстовый редактор тестирующей страницы.

Установка:
В браузере Настройки->Дополнительные инструменты->Расширения
(или открыть страницу chrome://extensions/)
В верхнем левом углу кнопка: "Загрузить распакованное расширение"->(Выбрать папку "Text Paster")
На панели инструментов браузера справа от адресной строки нажать кнопку "Расширения" и в строке "Text Paster" нажать иконку "Закрепить". В панели инструментов появится бледно-голубая кнопка с буквой "Т".
